## アプリ名

__Filmshare__

## 説明

映画、ドラマ、アニメの感想をシェアするSNSです。  
同じ作品を見た人や気になる作品の感想を知ることができます。

## 開発環境

* Ruby 2.5.7
* Ruby on Rails 5.2.4

## 使い方

使用するには会員登録が必要です。
1. http://54.178.207.83/ にアクセスする
2. ヘッダの新規登録をクリック
3. 会員登録を行う

投稿する
1. ヘッダの投稿をクリック
2. 投稿内容を記入、選択し送信をクリック

ユーザ情報編集
1. ヘッダのマイページをクリック
2. ユーザー情報編集ボタンをクリック
3. アイコン、ユーザ名、紹介文が編集可能
